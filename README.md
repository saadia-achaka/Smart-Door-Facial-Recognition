# Smart Door Facial Recognition System

## 📌 Description

Smart Door Facial Recognition System is an intelligent access control system based on computer vision and facial recognition.

The main objective of this project is to design a smart door system capable of identifying authorized users and improving security by replacing traditional access methods.

The system detects faces from images or video streams, extracts facial features, and compares them with a database of registered users.

---

## 🎯 Project Objectives

The main objectives of this project are:

- Detect human faces automatically.
- Extract facial features from detected faces.
- Recognize registered users.
- Provide an intelligent access control mechanism.
- Prepare the integration with an automatic door opening system.

---

## 🛠️ Technologies Used

- **Python** : Main programming language.

- **Haar Cascade** : Face detection algorithm.

## ⚙️ System Workflow

The system follows these main steps:

1. Image acquisition from camera or dataset.
2. Face detection using Haar Cascade.
3. Face feature extraction.
4. Comparison with registered faces.
5. Recognition decision:
   - Authorized person → Access granted.
   - Unknown person → Access denied.

---

## 🚀 Future Improvements

Possible improvements:

- Integration with Raspberry Pi.
- Servo motor control for automatic door opening.
- Face liveness detection to prevent spoofing attacks.
- Use of deep learning models such as FaceNet.
- Email or WhatsApp alerts for unauthorized access.
